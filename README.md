# kyucannes-events
# kyu @ Cannes Lions 2026

Internal hub showing kyu collective activity during Cannes Lions 2026.

## Run locally
npm install
npm run dev    # opens http://localhost:5173

## Edit the schedule
All events live in the `SEED` array near the top of `src/App.jsx`.
Edit / add / remove entries there and redeploy (push to GitHub → Vercel auto-builds).

## Build
npm run build  # outputs to /dist
